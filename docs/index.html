Dell Monitor Switch Powershell Script for U3415W and Dell Display Manager (DDM)

This project is a small powershell script to make it easier to switch between multiple monitors connected to a DDM supported monitor. I use it with my U3415W; but it should work with any dell monitor that supports the display manager. It was designed for use with the Dell U3415W 34" Ultrawide Monitor.

Pre-requisites

order to run this script the following Pre-requisites are required

  • This script is for Windows systems. In order for the script to work, Powershell should be enabled. This article describes how to set the execution policy of powershell and the execution polices are described here.
  • The script requires the Dell Display Manager (DDM)ddm.exe. DDM can be downloaded from Dell here.

Installation

Running Directly

To install the script, save switchdell.ps1 to a folder. I store my mine in \_scripts\ddm\switchdell.ps1

If execution policy permits, you can run it from the powershell command line.

Alternatively, a desktop shortcut can be created to run the script (this is how the author uses it)

Running from a Windows Desktop Shortcut

Create a shortcut

  • right click on the desktop. Select New->Shortcut
  • For the shortcut item (first screen of the wizard) select powershell.exe. On my system its in C:\Windows\System32\WindowsPowerShell\v1.0\powershell.exe, so that is what I enter in the box
  • Add parameters to the shortcut item to invoke switchdell.ps1. So the full entry for the shortcut is C:\Windows\System32\WindowsPowerShell\v1.0\powershell.exe -command "&C:\_scripts\ddm-ctrl\switchdell.ps1 -Monitor HDMI" is what I used to swtich to the HDMI input from mindi Display port
